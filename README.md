## Project Overview
This project is an **Outfit Matcher App** that helps users create outfits from the clothes they already own.  
The app uses **object-oriented design principles** to organize clothing items (like tops, pants, and shoes) and suggest combinations based on matching colors, categories, and seasons.

---

## Planned Features
- Add garments (name, type, color, season)
- Store wardrobe data
- Match compatible outfits automatically
- (Optional) Recommend based on weather or occasion
- Display suggested outfits
